# ScaleMed Frontend

Welcome to the ScaleMed Frontend repository! This is the frontend component of the ScaleMed App, a ToDo application that allows users to manage their tasks efficiently. The frontend is built using Next.js, React, and interacts with the ScaleMed Backend API.

## Prerequisites

Before getting started with the ScaleMed Frontend, ensure you have the following prerequisites installed on your system:

- Node.js (v14 or later)
- npm (Node Package Manager)

## Installation

To install and run the ScaleMed Frontend, follow these steps:

1. Clone the repository:
 ```git clone https://github.com/gpadova/scaleMedFrontend.git```


2. Navigate to the project directory:
```cd scaleMedFrontend```

3. Install the dependencies:
```npm install```

4. Start the server:
 ```npm run build && npm run start```

## Usage

Once the frontend is running, open your web browser and access `http://localhost:3000` to use the ScaleMed App. You can create tasks, mark them as concluded, and delete them using the provided user interface.

## Folder Structure

The main folders and files in the ScaleMed Frontend project are organized as follows:

- `src/app`: Contains the pages of the application.
- `src/components`: Contains reusable React components used throughout the application.

Feel free to explore the codebase and customize the frontend as per your requirements.
